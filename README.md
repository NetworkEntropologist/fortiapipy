`fortiapipy` is a work-in-progress Python API client for Fortinet products. 

At a high level, this project acts as a wrapper and API client for the extensive
REST API offered by most Fortinet products. Currently, only supporting FortiOS as 
run on Fortigate hardware and software firewall and network securoty appliances, 
but support for more products in the family wll be added in due course.

## Project Status: Work in Progress

This project is under active development and should currently be considered
experimental. Interfaces, behavior, and internal structure are likely to change.

## Development setup

This project uses pre-commit.

Install hooks with:
`pre-commit install`
